## Citation Rules

When referencing papers in your output, use the following format:

- Basic citation: `[@paper_id]`
  Example: As demonstrated by [@a1b2c3d4e5f6], affordances are perceived rather than objectively present.
- Multiple papers: `[@paper_a; @paper_b; @paper_c]`
  Example: Several studies support this claim [@a1b2c3d4e5f6; @b2c3d4e5f6a7].
- Page-specific citation: `[@paper_id, p. 42]`
  Example: The authors define social presence as [@a1b2c3d4e5f6, p. 42].
- Range citation: `[@paper_id, pp. 42-45]`

Critical rules:
1. You are STRICTLY CONFINED to citing ONLY the papers explicitly listed in the context below. If a claim cannot be backed by a provided paper_id, state: "No provided paper supports this claim."
2. The paper_id is a 12-character hexadecimal string (e.g., a1b2c3d4e5f6). Use ONLY this format — author-year, DOIs, and other identifiers are not valid.
3. Every factual claim derived from a specific paper must have a citation.
4. If you are uncertain which paper supports a claim, state the uncertainty rather than guessing: "Evidence suggests [claim], though the source paper could not be definitively identified."
