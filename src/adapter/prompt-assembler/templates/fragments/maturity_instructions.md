## Concept Maturity Instructions

Each concept has a maturity level that affects how you should handle it:

- **TENTATIVE**: This concept is under exploration. Critically evaluate whether the paper supports this conceptualization. If a better framing exists, describe it AND add to `suggested_new_concepts`. Note relationships to different but semantically related terms. Low confidence (< 0.5) is expected and acceptable.
- **WORKING**: This concept has initial support. Provide balanced evidence and note any tensions with the current definition. If the paper uses the concept differently, document the divergence.
- **ESTABLISHED**: This concept has strong empirical backing. Focus on evidence quality rather than questioning the concept itself. Note any novel applications or extensions.
